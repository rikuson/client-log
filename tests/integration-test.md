# Integration Test

`npm run test`を実行して確認する。  

- デフォルトメソッド
  - [ ] ログの先頭に現在時刻が表示されること
  - [ ] ラベルが正しく表示されること
  - [ ] ラベルの色が正しく表示されること
  - [ ] メソッドを呼び出した箇所の行数が表示されること
  - [ ] `error`メソッドで`console.error`が呼び出されること
- ユーザー定義メソッド
  - [ ] ログの先頭に設定したフォーマットで現在時刻が表示されること
  - [ ] 設定したラベルが表示されること
  - [ ] ラベルを設定しなかった場合はメソッド名がラベルとして表示されること
  - [ ] ラベルが設定した色になっていること
